Description
------------------------------------------------------------------

This is my Neovim configuration. I use mini.nvim from Evgeni
Chasnovski as a base for my configuration. I also use the great
Neovide GUI integration of neovim.

By cloning this repository you will get:
- a working LSP system (autocompletion and more)
- a working DAP system (debuging)
- configured keymappings
- configured neovide settings


Installation
------------------------------------------------------------------

You only need to clone this repository to your nvim config folder.

In `~/AppData/Local/nvim` for Windows and `~/.config/nvim` for
Linux.
